Introduction

EbonFarm is a graphical client for the telnet version of Nethack. Normal telnet clients only support text interfaces. EbonFarm maps different character and color combinations to Nethack images, and can accept tile data from the server. This manual describes the EbonFarm client. For information on the actual Nethack game, see one of the many sources online.


Features

  • Graphics Mode: Most telnet characters are replaced with a descriptive image from one of several tilesets. You can switch between graphics and text mode with the Home key.

  • Server Tile Data: Some servers support the 'Glyphs' extension, which tells the client exactly which image to use for each telnet character. This provides 100% accurate matching of graphics to characters. To enable the extension:

    • Log in to the server.
    • Edit the options file from the server's main menu.
    • Add OPTIONS=vt_tiledata to the file.

  • Tilesets: EbonFarm supports multiple tilesets, which can be found in the EbonFarm->Graphics Settings... menu.

  • Arrow Keys: The arrow keys can be used to control character movement.

  • Command Menus: There's a menu item for every Nethack command, along with its keyboard shortcut.

  • Information: Shift-clicking in the map returns the name of the creature, item, or dungeon feature selected.

Troubleshooting

Q: Some characters don't have any graphics!

A: Unfortunately, not all characters can be matched to a graphic. Some characters represent more than one type of thing, and no single graphic makes sense. For example, a white # can be either a sink or a corridor.

If you find a character that should have a graphic, but doesn't, you can try shift-right clicking on it to autodetect one.


Q: The walls and floors have the wrong graphics!

A: Try disabling DEC graphics. Go to Game->Options and change DECgraphics to false. You can use space to navigate through the options.


Q: Graphics mode is turned on/off at incorrect times.

A: EbonFarm does its best to figure out if the screen is filled with text or a dungeon level, and switches between text and graphics automatically. However, there are times when it picks the wrong one. You can manually change this using the Home key, or 'Graphics Enabled' from the Game menu.


Q: The arrow keys don't move my character.

A: Nethack doesn't normally accept arrow keys, and movement commands are either ykulnjbh or 12346789. EbonFarm converts the arrow keys to the appropriate movement commands, and sends them to the server. It could be that the server is set to accept number pad directions, and EbonFarm is sending letters, or vice-versa.

To change the commands the server accepts, go to Game->Options and select 'number_pad'. You can use space to navigate through the options.


Q: EbonFarm gives an error when I try to watch other players.

A: This is a known bug when watching players whose telnet window is larger than 80x24. Sadly, there is no fix in the forseeable future.


Q: I can't edit my configuration file online.

A: This is a known problem. While I have tried to incorporate as many xterm commands into EbonFarm as possible, it only contains a subset of them. You may want to try using a telnet program to edit config files.


Q: I've found a bug!

A: You can report bugs in base EbonHack to matthew.carlson@gmail.com. Please include your operating system, EbonFarm version, and the address of the server you were connected to.


Q: The Farming Window never activates!

A: EbonFarm requires the server to support VT_TILEDATA,
this is a conscious decision, as the tile-data includes an "End of Data" Message.
Without this message, EbonFarm cannot reliably determine when it is safe to send more data.
